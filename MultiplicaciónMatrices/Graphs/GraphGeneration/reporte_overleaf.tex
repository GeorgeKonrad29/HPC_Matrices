\section{Objetivo}
Este informe presenta el análisis de tiempo de ejecución y speedup para las variantes de multiplicación de matrices incluidas en el archivo de datos \texttt{resultados.csv}. Las gráficas se incluyen como evidencia visual del comportamiento observado.
\section{Marco Conceptual}
La métrica de \textit{wall-clock time} corresponde al tiempo real transcurrido durante la ejecución de un algoritmo. En este estudio se compara dicha métrica entre implementaciones secuenciales y paralelas (hilos y procesos) para multiplicación de matrices cuadradas.
El \textit{speedup} se define como $S = T_{base} / T_{paralelo}$, donde $T_{base}$ es el tiempo de la implementación de referencia (\texttt{standar.doc}) y $T_{paralelo}$ el tiempo promedio de la variante comparada para un mismo tamaño de matriz.
Dado que el costo de la multiplicación clásica de matrices crece aproximadamente como $O(n^3)$, se espera un incremento pronunciado en el tiempo al aumentar el tamaño de entrada.
